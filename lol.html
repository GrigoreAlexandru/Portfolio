About

There are 3rd party apps like op.gg or lolnexus that show info about your in-game teammates , but I always wanted to make my own league related app using the Riot API so that's how this app came to be . At the current moment I think it looks better than lolnexus , which was on of the major point , I still want to redo the table design and make it align horizontally. Check it out here

Note: This uses dev key from Riot API and is not for public use , showcase only . Thus it is required to regenerate the API key everyday manually, which I may or may not do .
