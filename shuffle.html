YouTube Shuffle

Includes PHP, JavaScript/jQuery and Codeigniter

Introduction

YouTube shuffle only shuffles 200 videos, if your music playlist has more than 200 items then you don't have any choice but to use 3rd party apps . There is one such app but takes a lot of time to load large playlists , and it's pretty ugly .

Initial Ideas

Not wanting to use ugly the 3rd party app I decided to make my own. Even though this could be done way more easily with just JavaScript , I wanted to try a PHP MVC framework and work around the MVC concept so I chose Codeigniter for its ease of use. One other big reason to use PHP was security , the 3rd party app mentioned above has its API key and client id written in js , basically plain sight and in theory, anybody could use them to ddos the YouTube API .

Final Concept

Everything was arranged , all that was left was to write the app itself.After finishing writing the app , I needed the UI . I originally intended to write the UI from 0 using Material Design but thankfully Google offers templates which can be found at the Material Design Lite website. I chose one of the templates avalaible , modified it and wrote the remaining PHP and JavaScript code. There was a issue with Iframe API which made autoplaying the next videos in queue unavailable , thankfully that was fixed and now everything works .
